# GoTrustWDU Token (TUR)

GoTrustWDU Token (TUR) is a clean, transparent, and fixed-supply BEP-20 utility token deployed on the BNB Smart Chain.

The project is designed with simplicity, verifiability, and predictable on-chain behavior in mind, making it suitable for basic Web3 utilities, community systems, and educational blockchain use cases.

---

## Token Information

- Name: GoTrustWDU Token  
- Symbol: TUR  
- Decimals: 18  
- Total Supply: 9,000,000,000,000 TUR  
- Standard: BEP-20 (ERC20 compatible)  
- Network: BNB Smart Chain  

---

## Contract Features

- Fixed total supply minted at deployment  
- No mint function  
- No burn function  
- No transaction tax or fees  
- No blacklist, pause, or hidden owner functions  
- Fully verified source code on BscScan  

The smart contract is intentionally minimal and transparent.

---

## Use Cases

GoTrustWDU Token is intended for:

- Community reward and participation systems  
- Educational blockchain demonstrations  
- Lightweight Web3 utilities  
- Transparent on-chain interactions  

This token is not designed as an investment product.
