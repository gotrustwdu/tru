About the Project

GoTrustWDU (TUR) is a transparent and fixed-supply BEP-20 utility token deployed on the BNB Smart Chain. The project focuses on simplicity, verifiability, and responsible on-chain usage.

The smart contract is fully verified and does not include minting, burning, taxation, or privileged administrative functions. All token behavior can be independently verified on-chain.

Project Principles

Transparency

The source code is publicly available and verified on BscScan, allowing anyone to audit and review the contract logic.

Predictability

GoTrustWDU has a fixed total supply with no mint or burn mechanisms, ensuring predictable token behavior over time.

Utility-Oriented

The token is designed for basic Web3 utilities, educational use cases, and lightweight blockchain integrations.

Team & Governance

GoTrustWDU is a community-driven utility token project. The project does not operate with a publicly promoted founding team or investment entity.

Maintenance and updates are carried out transparently through on-chain verification, public documentation, and openly accessible project resources.
